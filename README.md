## Usage

Basically, you open the python script. It gives you a list of non-pro locations, which you can connect to by selecting in fzf interactively. You can also choose pro if you want using an argument.
The help screen basically explains it all.

```text
 Usage: wsconnect [OPTIONS]

 Interactively select and connect to a Windscribe location using fzf.

╭─ Options ────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --pro                               Include Pro locations in the list                                                                │
│ --filter              -f      TEXT  Pre-filter locations                                                                             │
│ --install-completion                Install completion for the current shell.                                                        │
│ --show-completion                   Show completion for the current shell, to copy it or customize the installation.                 │
│ --help                              Show this message and exit.                                                                      │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
```
